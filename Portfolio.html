Dean Rice's Portfolio

My latest project is a scheduling software application that I have been developing in conjunction with the UNF English department and UNF Writing Center. As of right now, the UNF English department's course scheduling process for each semester is a time and labor intensive manual process. Consequently, each semester several English faculty must exert time and energy using this manual process to match English professors to appropriate English courses. My goal with this project is to develop a software solution that eliminates much of the time and labor of the current scheduling process by utilizing computation to execute the matching portion of the scheduling process. In order to accomplish this goal I designed an application that will allow a user to upload an excel file (the format requested by my colleagues) that contains the data for which courses will be offered in the coming semester. The software uses this data set in conjunction with a second data set (also an excel file) that contains information corresponding to the course preferences of the English professors that will be teaching courses for the semester. The user will then be able to run the software, creating a one semester schedule in which all the courses to be offered have been matched to an appropriate professor(appropriateness is based on professor's preferences and qualifications). So far, I have implemented the majority of the code for the back-end, with only a few methods and one class to code over the coming weeks to be done with that part of the software. I am also currently learning how to use Java Fx so that I can begin developing the UI after I finish the back-end. I hope to have a testable and demoable product in the coming months!
